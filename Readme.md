Planificador de Sesiones de Estudio
Descripción

Este proyecto es una página web interactiva que permite organizar sesiones de estudio donde el usuario puede agregar materias, temas, duración de estudio y prioridad y las sesiones se muestran en una tabla dinámica donde también se pueden marcar como completadas o eliminarlas.

El objetivo principal es ayudar a organizar el tiempo de estudio de forma visual y sencilla.

Estructura del proyecto

El proyecto está compuesto por los siguientes archivos principales:

index.html

Contiene la estructura principal de la página web, incluyendo el formulario para agregar sesiones de estudio, la tabla donde se muestran los datos y las imágenes.

styles.css

Archivo encargado del diseño visual de la página. Incluye colores, fuentes, estilos del formulario, tabla y diseño responsivo.

script.js

Contiene la lógica del proyecto. 

Permite:Capturar los datos del formulario.

Validar los campos.

Agregar sesiones a la tabla.

Eliminar sesiones.

Marcar sesiones como completadas.

Calcular estadísticas como total de sesiones y horas de estudio.

Tecnologías utilizadas

HTML5

Para la estructura semántica de la página.

CSS

Para el diseño visual, uso de variables CSS, estilos responsivos y animaciones.

JavaScript

Para agregar interactividad al formulario y gestionar los datos dinámicamente.

Instrucciones de uso

Abrir el archivo index.html en cualquier navegador web.


Completar el formulario con los datos de la sesión de estudio:

Materia: nombre de la materia.

Tema: contenido o tema a estudiar.

Horas: duración estimada de la sesión.

Dificultad: nivel de dificultad del tema.

Prioridad: elegir entre importante, normal o repaso.

Fecha: día planificado para estudiar.

Presionar el botón Agregar para guardar la sesión de estudio.

La sesión aparecerá en la tabla de sesiones, donde se podrá:

Marcar la sesión como completada.

Eliminar la sesión si ya no es necesaria.

En la parte inferior se mostrará un resumen automático con:

Total de sesiones agregadas.

Total de horas de estudio.

Cantidad de sesiones importantes.

También se puede usar el campo de búsqueda por materia para filtrar las sesiones registradas.
